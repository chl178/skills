# Skills

This repository contains two Codex/Codex CLI skills:

- `ralphinho-rfc-pipeline/` — RFC-driven multi-agent DAG execution pattern with quality gates, merge queues, and work unit orchestration.
- `sonic-architecture/` — Design and implement SONiC switch software features across its containerized, Redis/SAI-driven architecture.

Each skill has its own `SKILL.md` at the path below:

- `ralphinho-rfc-pipeline/SKILL.md`
- `sonic-architecture/SKILL.md`

## Usage
These skills are designed to be installed into a Codex environment. Copy a folder into your Codex skills directory (e.g., `$CODEX_HOME/skills`) or point your tooling at these paths.
